# Imágenes de juegos

Carpeta para banners e imágenes visuales de cada juego de LenguArcade.

## Formato recomendado

- Banner horizontal: 1200 x 675 px, formato PNG o JPG.
- Nombre de archivo estable, sin espacios ni tildes.
- Peso ideal: menos de 500 KB por imagen.

## Archivos previstos

```text
battlegrafia-banner.png
maniacgrafia-banner.png
narratoria-banner.png
versopolis-banner.png
scrabble-banner.png
conjuga-apuesta-banner.png
verb-battle-banner.png
```

## Cómo usarlas después

En la hoja `Juegos`, columna `banner`, podremos guardar la URL pública de cada imagen o una clave interna. Lo más cómodo será subirlas a GitHub Pages o a Google Drive con enlace público.
